## Contoh dari UI yang kurang bagus

Dalam merancang sebuah web tentu kita tidak lepas dari desain dari sebuah antarmuka atau UI,UI digunakan oleh user sebagai alat untuk berdialog dengan komputer,oleh karena itu baik buruk UI sangat mempengaruhi user dalam melakukan kegiatan dengan komputer

Berikut contoh UI yang aneh pada input nomor telepon dengan menggunakan slider untuk input angka:

## [Klik disini untuk demo](https://goulartnogueira.github.io/BadUI/Phone-Slider-Selector/BadUIPhone.html)

![Pemandangan Pantai](https://github.com/Jokeronii/BadUI/raw/master/Phone-Slider-Selector/BadUI%20Phone%20Selector.gif)

Contoh berikutnya adalah form untuk input tanggal lahir tetapi menggunakan tombol prev dan next sebagai input angka,tidak hanya itu bahkan terdapat fitur tambahan untuk seberapa jauh untuk melakukan prev dan next sebagai input.
